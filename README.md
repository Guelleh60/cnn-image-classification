# CNN Image Classification

## Description
This project implements a Convolutional Neural Network (CNN) to classify images from the CIFAR-10 dataset.

## Objectives
- Understand image classification
- Build a CNN model using TensorFlow
- Evaluate model performance

## Tools & Technologies
- Python
- TensorFlow / Keras

## Model Architecture
- Conv2D layers
- MaxPooling layers
- Fully connected (Dense) layers

## Results
The model achieves approximately 68% accuracy on the test dataset.

## How to Run
1. Open the notebook in Google Colab
2. Run all cells step-by-step
